# Clicker PoC-Idle
## Description
Little web/mobile game with idle and click mechanics. And also with a modular system for improving game mechanics.
Game including MVVM+R3 modular architecture.

## Controlls
Just clicks and upgrades, no more things.

## Architecture
Game based on MVVM pattern + R3 for UI and reactive update game state/flow. The project also uses UniTask for idle mechanics. A simple self-written DI framework is used to resolve dependencies (it is planned to switch to VContainer in the future)
